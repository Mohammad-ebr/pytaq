Effective Spreads

DataFrame

TaqDaliy.compute_effective_spreads(date=None, symbols=None, trade_and_nbbo_df=None)

Accept Trade-NBBO DataFrame, calculate Dollar Effective Spreads and Percent Effective Spreads, remove crossed and locked quotes, and return as Effetive Spreads DataFrame for given symbols at specific date.

Parameters

  • date (datetime instance) – Day that market was open at it.

  • symbols (list[str]) – Symbols that data is related to it.

  • trade_and_nbbo_df (Pandas DataFrame) – If trade_and_nbbo_df is provided , calculate Effective Spreads DataFrame from it as a local file, otherwise get it directly from database.

Returns

Effetive Spreads.

Return type

Pandas DataFrame

Example 1: Calculate Effetive Spreads DataFrame by providing date and symbols.

from datetime import datetime
from pytaq import TaqDaily
import wrds

# connecting to WRDS database, use your username instead of 'username'
db = wrds.Connection(wrds_username='username')
taq = TaqDaily(method='PostgreSQL', db=db)

# getting Effective Spreads DataFrame using taqdaily instance
eff_spreads_df = taq.compute_effective_spreads(datetime(2016,12,7), ['IBM'])
eff_spreads_df

Example 2: Calculate Effetive Spreads DataFrame by providing trade_and_nbbo_df local file.

from datetime import datetime
from pytaq import TaqDaily
import wrds

# connecting to WRDS database, use your username instead of 'username'
db = wrds.Connection(wrds_username='username')
taq = TaqDaily(method='PostgreSQL', db=db)

# getting Trade-NBBO table, but you can use local files
trade_and_nbbo_df = taq.merge_trades_nbbo(datetime(2016,12,7), ['IBM'])

# getting Effective Spreads DataFrame using taqdaily instance
eff_spreads_df = taq.compute_effective_spreads(trade_and_nbbo_df=trade_and_nbbo_df)
eff_spreads_df
