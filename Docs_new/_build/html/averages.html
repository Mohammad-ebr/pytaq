Averages

TaqDaliy.compute_averages_ave_sw_dw(df, measures, simple=True, dollar_weighted=True, share_weighted=True)

Accept a DataFrame, calculate selected kind of averages for selected measures(columns).

Parameters

  • df (Pandas DataFrame) – DataFrame that has measures to calculate averages.

  • measures (list[str]) – Measures(columns) to calculate averages.

  • simple (bool, default True) – If simple=True, calculate simple averages for all measures with suffix ‘_Ave’

  • dollar_weighted (bool, default True) – If dollar_weighted=True, calculate dollar_weighted averages for all measures with suffix ‘_DW’

  • share_weighted (bool, default True) – If share_weighted=True, calculate share_weighted averages for all measures with suffix ‘_SW’

Returns

Some kind of averages.

Return type

Pandas DataFrame
